Caesar

Caesar "C"

For this problem set, our task was to create a program that would encrypt messages.

I admit that the logic seemed simple enough, however putting that into practice proved difficult.

I needed the program to a=promt the user for input and only accept an input that is not negative. I also had to make sure that it would not a bump up the character more than 26 if a higher number was inputted.

With the input I then had to prompt the user to input plaintext. I would then use the key to encrypt the message, and output the ciphertext.
