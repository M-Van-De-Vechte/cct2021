Linear Search

What is Linear Search?

Linear Search goes through and hecks every item in the list.

This search is not an efficient search method becuase at the worst case it would go through the entire list.

Bubble Sort

What is Bubble Sort?

Bubble Sort is a sorting method which goes through a list of items and compares two side by side and sees if they are in order. If they are not in order than the items are switched. This continues on until the list is in order.

In this lab we had to finish writing the bubble sort function sop that it sorts the list correctly.

Selection Sort

What is Selection Sort?

Selection sort is a sorting method that goes though the entire list and finds the smallest number. It then swap it with the number at the front and then continues on until the list is sorted.

In this lab we had to finish the function so that it was sorting properly.

Insertion Sort

What is Insertion Sort?

Insertion sort is a sorting method that works by dividing the array into two parts sorted and unsorted.

It goes through the list and finds a number that can be inserted in the array so that it is sorted.

In this lab we had to complete the function so that it sorts the array properly.

What is Binary Search

Binary Search works by dividing the array into two, and then again, and agaiin until it finds the number.
